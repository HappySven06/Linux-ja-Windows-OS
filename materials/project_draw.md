# Projekti loosimine

Iga meeskond loosib oma ettevõtte IT infrastruktuuri lähteandmed.

## Reeglid

1. Iga meeskond veeretab täringut 3 korda:
   - kasutajate arv
   - ettevõtte tüüp
   - eritingimus

2. Tulemused kirjutatakse üles.

3. Meeskond võib **ühe korra uuesti veeretada (ükskõik millise kategooria)**.

---

## 1. Kasutajate arv

Valem:
> kasutajad = baas + täringu tulemus (1–6)

### Baas vastavalt meeskonna suurusele:

| Meeskonna suurus | Baas |
| ---------------- | ---- |
| 2 õpilast        | 8    |
| 3 õpilast        | 12   |
| 4 õpilast        | 15   |


---

## 2. Ettevõtte tüüp

| Täring | Ettevõte              |
| ------ | --------------------- |
| 1      | Veebiarendus ettevõte |
| 2      | Mängustuudio          |
| 3      | Raamatupidamisfirma   |
| 4      | Startup               |
| 5      | Kool / koolituskeskus |
| 6      | E-pood                |

---

## 3. Eritingimus

| Täring | Tingimus                     |
| ------ | ---------------------------- |
| 1      | Kaugtöö (VPN vajalik)        |
| 2      | Kõrge turvalisus             |
| 3      | Failijagamine on kriitiline  |
| 4      | Varukoopiad on kohustuslikud |
| 5      | Piiratud eelarve             |
| 6      | Kiire kasv (skaaleerimine)   |
