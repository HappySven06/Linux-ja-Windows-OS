# Projekt: IT infrastruktuuri disain ja teostus

## Projekti kirjeldus

Teie olete IT-meeskond, kelle ülesanne on luua ettevõtte IT infrastruktuur.

Projekt hõlmab:
- infrastruktuuri planeerimist
- süsteemide seadistamist
- dokumenteerimist

---

## Meeskonna moodustamine ja kinnitamine

Iga meeskond valib endale ühe otsustava isiku.

Otsustava isiku ülesanded:
- Loosida meeskonnale senaarium
- Luua ühine GitHub repo.
- Kinnitada enda meeskond ja senaarium

### Senaariumi loosimine

Meeskonna otsustav isik loosib enda meeskonnale senaariumi kasutades loosisüsteemi kirjeltatud siin: [Täringu loos](project_draw.md).

**Stsenaarium koosneb:**
- Kasutajate arv (määrab infrastruktuuri skaleerimise vajaduse)
- Ettevõtte tüüp (määrab rakendused ja teenused)
- Eritingimus (määrab spetsiaalsed nõuded)

### Meeskonna ja senaariumi kinnitamine

Meeskonna otsustav isik saadab õpetajale kirja kokkulepitud keskkonnas (nt Discord, email) järgneval kujul:

```
<PLATVORMI_TERVITUS> Meeskond <meeskonna number>

Meeskonna senaarium:
<kasutajate arv>, <ettevõtte tüüp>, <eritingimus>

Meeskonna liikmed:
<liikme nimi> - Otsustav isik
<liikme nimi>
...

Meeskonna GitHub
<GitHub repo link>
```

**Näited:**
- Discord: `@teacher Meeskond 1`
- Email: `Meeskond 1`

**GitHub repo:**
Looge avalik GitHub repo ning lisage see kirjale.
