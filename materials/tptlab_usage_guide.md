# TPTLab

TPTLab on kooli poolt loodud keskkond, kus õpilased saavad luua õppetöö jaoks virtuaalmasinaid.

# Ühendamine

TPTLab keskkonnale ligipääsemiseks tuleb esmalt ühendada TPTLab VPN-iga.

Selleks on vaja kasutada mõnda OpenVPN-i rakendust (soovitan kasutada OpenVPN Connecti).  
Teile on saadetud e-posti teel ühenduse profiili/configuratsiooni fail (.ovpn laiendiga).

VPN-i sisselogimisel tuleb kasutada oma kooli e-posti aadressi ilma `@techno.ee` lõputa.  
Näiteks: `john.doe@techno.ee` → `john.doe`

Parooliks on sinu kooli e-posti parool.

> **Märkus:** On võimalik, et esimesel ühendamise katsel see ei õnnestu.  
> Sel juhul lahendab probleemi tavaliselt kooli e-posti parooli vahetamine.

# Kasutamine

Süsteemile ligipääsemiseks ava veebileht **vcenter.tptlab.eu** ning logi sisse.

Sisselogimisel kasuta oma kooli e-posti aadressi ja parooli.

Pärast sisselogimist leiad vasakult menüüst kirje **vcenter.tptlab.eu**, mille kõrval on väike nool.  
Sealt tuleb navigeerida oma profiilini.

Tavaliselt näeb teekond välja umbes selline:  
`vcenter.tptlab.eu → TPTLAB → TPTLAB → student → rp_(klassi_kood) → rp_(sinu_nimi)`

Sealt leiad sulle loodud virtuaalmasinad.
